# Failnaught
A study app for tracking hours, content, and building long-term memory.

Failnaught uses the SQLite library as its database. The resources used
so far to implement the SQLite database are:

- "Documentation" by SQLite.org: 
https://sqlite.org/docs.html

- "SQLite Introduction" by Caleb Curry on Youtube: 
https://www.youtube.com/watch?v=8Xyn8R9eKB8

- "Command Line Shell For SQLite" by SQLite.org: 
https://sqlite.org/cli.html

- "SQL As Understood By SQLite" by SQLite.org: 
https://sqlite.org/lang.html

- "Datatypes In SQLite" by SQLite.org: 
https://sqlite.org/datatype3.html
